# Machine Learning Repository

## Overview

This repository contains machine learning projects and algorithms implemented in Python using Jupyter Notebooks. It covers various ML concepts including regression, classification, and data analysis.

## Projects Included

### 1. **Multiple Regression Analysis**
- `ml.ipynb` - Core machine learning concepts and implementations
- `multiple_reg.ipynb` - Multiple linear regression models
- `multi_reg_advertising.ipynb` - Regression analysis on advertising dataset

### 2. **Insurance Prediction Models**
- `insurance_multi_reg.ipynb` - Insurance premium prediction using multiple regression
- `insurance.csv` - Insurance dataset for model training and evaluation

### 3. **Data Processing**
- `50_Startups.csv` - Startup revenue prediction dataset
- `Salary_Data.csv` - Employee salary analysis dataset
- `advertising.csv` - Advertising spend and sales data

## Technologies Used

- **Python 3.x**
- **Jupyter Notebook** - Interactive development environment
- **NumPy** - Numerical computing
- **Pandas** - Data manipulation and analysis
- **Scikit-learn** - Machine learning algorithms
- **Matplotlib & Seaborn** - Data visualization

## Key Concepts Covered

✓ Data Preprocessing and Exploratory Data Analysis (EDA)
✓ Linear Regression & Multiple Regression
✓ Feature Scaling & Normalization
✓ Train-Test Split & Cross-Validation
✓ Model Evaluation Metrics (R², MSE, RMSE)
✓ Visualization & Interpretation

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Abhinav7301/Machine_Learning.git
   ```

2. Install required packages:
   ```bash
   pip install numpy pandas scikit-learn matplotlib seaborn jupyter
   ```

3. Open Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

4. Explore the notebooks to understand the implementations and modify as needed.

## Dataset Information

- **50_Startups.csv** - Revenue prediction for startups
- **Salary_Data.csv** - Employee salary based on experience
- **advertising.csv** - Sales prediction from advertising spend
- **insurance.csv** - Insurance premium estimation
